This project builds an WPF desktop application that is able to download radio/tv broadcasts from Danmarks Radio through publicly available ressources i the danish national 
library at url: https://www.kb.dk/find-materiale/dr-arkivet/

As a prerequisite to make this work you have to install ffmpeg first from https://ffmpeg.org/download.html and put it in your path. So add c:/ffmpeg/bin to path variable 
after install.
