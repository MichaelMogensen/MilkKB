This project builds an WPF desktop application that is able to simultaneously download 10 radio/tv broadcasts from Danmarks Radio through publicly available ressources at the danish national library at url: https://www.kb.dk/find-materiale/dr-arkivet/

As a prerequisite to make this work you have to install ffmpeg first from https://ffmpeg.org/download.html and put it in your path. So add c:/ffmpeg/bin to path variable 
after install.

Also you have to have Chrome installed at its default location: C:\Program Files\Google\Chrome\Application\Chrome.exe before running it for the first time.
